# 快速开始

- 通过框架模版快速初始化项目

## 设置 npm 镜像源

```js
// npm
npm config set registry https://registry.npmmirror.com
// yarn
yarn config set registry https://registry.npmmirror.com
```

## 通过模版快速初始化

- 通过模版快速初始化

```js
// 拉取模版
git@github.com:yunke-yunfly/YUNFLY-TEMPLATE.git

// 安装依赖
yarn install

// 启动项目
yarn dev
yarn watch:dev

// open
http://127.0.0.1:3000/example/simple/get?name=xxx
```

## 生产环境

```js
// 编译
yarn build

// 运行
yarn run run
```
